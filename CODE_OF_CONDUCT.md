# Code of Conduct

This project aims to be practical, respectful, and welcoming.

## Expected behavior

- Be kind and direct.
- Assume good intent while still being clear about quality and safety concerns.
- Critique ideas, instructions, code, and examples rather than people.
- Respect different experience levels with Codex, frontend design, and open
  source workflows.

## Unacceptable behavior

- Harassment, insults, threats, or discriminatory language.
- Sharing private information without permission.
- Repeatedly derailing technical discussions.
- Contributing malicious or misleading skill behavior.

## Enforcement

Maintainers may edit, reject, or remove contributions that violate this code of
conduct. Serious or repeated violations may result in being blocked from
participating in the project.
